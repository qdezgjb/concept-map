# 概念图自动生成系统

基于AI的智能概念图生成与编辑平台，支持关键词生成、文本分析、交互式编辑等功能。

## 功能特性

### 🚀 核心功能
- **关键词生成概念图**: 输入一个关键词，AI自动生成相关概念图
- **文本分析生成概念图**: 输入一段描述文字，AI分析并生成概念图
- **交互式编辑**: 支持拖拽、编辑、添加/删除节点和连线

### ✏️ 编辑工具
- **节点操作**: 添加、删除、编辑节点
- **连线操作**: 添加、删除、编辑连线
- **布局控制**: 力导向、层次、圆形等多种布局算法
- **样式设置**: 节点颜色、形状，连线样式等

### 💾 数据管理
- **保存/加载**: 支持JSON格式的概念图文件
- **撤销/重做**: 操作历史记录管理
- **导出功能**: 支持多种格式导出

## 技术架构

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **可视化**: D3.js (计划集成)
- **AI集成**: OpenAI GPT-4 / Claude API (计划集成)
- **布局算法**: 力导向、层次、圆形等算法

## 快速开始

### 1. 打开系统
直接在浏览器中打开 `web/index.html` 文件即可使用。

### 2. 生成概念图

#### 方式一：关键词生成
1. 在左侧"关键词生成概念图"区域输入中心概念
2. 点击"生成概念图"按钮
3. 系统将自动生成相关概念图

#### 方式二：文本分析
1. 在左侧"文本分析生成概念图"区域输入描述文字
2. 点击"分析生成"按钮
3. 系统将分析文本内容并生成概念图

### 3. 编辑概念图
生成概念图后，可以使用右侧的编辑工具栏：
- 添加/删除/编辑节点和连线
- 调整布局算法
- 修改颜色和样式

### 4. 保存和导出
- 使用状态栏的保存按钮保存为JSON文件
- 使用导出按钮导出为图片格式

## 使用说明

### 界面布局
- **左侧**: 功能选择区域，包含生成和编辑功能
- **右侧**: 概念图展示和编辑区域
- **顶部**: 系统标题和说明
- **底部**: 系统版本信息

### 操作流程
1. **输入内容** → 选择生成方式并输入关键词或描述
2. **生成概念图** → 系统调用AI分析并生成概念图
3. **编辑优化** → 使用编辑工具调整节点位置和样式
4. **保存导出** → 保存工作结果或导出为其他格式

### 快捷键
- **Enter**: 关键词输入框快速生成
- **Ctrl+Enter**: 描述文本框快速生成

## 注意事项

1. **AI功能**: 目前使用模拟数据，真实AI集成将在后续版本中实现
2. **数据存储**: 概念图数据仅保存在浏览器内存中，刷新页面会丢失
3. **文件格式**: 保存的概念图使用JSON格式，确保数据完整性
4. **性能考虑**: 大型概念图可能影响渲染性能，建议节点数量控制在100个以内
